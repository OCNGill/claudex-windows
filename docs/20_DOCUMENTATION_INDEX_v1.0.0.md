## Glossary

### Key Terminology

**Claudex:** AI-assisted code execution and workflow automation system

**Session:** A stateful execution context with configuration, hooks, and history

**Hook:** Extensibility mechanism for executing custom code at various points

**Profile:** Pre-configured templates for common workflows

**MCP:** Model Context Protocol for AI model integration

**Launch Mode:** 
- **NEW:** Create completely new session
- **RESUME:** Continue existing session
- **FORK:** Create variant of existing session
- **FRESH:** New session with current project state
- **EPHEMERAL:** Temporary session that doesn't persist
